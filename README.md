## featured projects

### [Touka / とうか](https://github.com/pratiksingh94/touka) — Browser-based Packet Analyzer
> Built a Wireshark-style network traffic analysis tool that runs entirely in the browser. Zero external parsing libraries, binary PCAP parsing, protocol dispatch, and dissectors for Ethernet, IP, TCP, UDP, ICMP, ARP, etc all written from scratch in TypeScript. Your captures never leave your machine.

### [Mesh-C2](https://github.com/pratiksingh94/mesh-c2) — P2P Mesh C2 Framework
> C2 framework where implants gossip commands peer-to-peer across a mesh. The central server can die and ops keep running. Built in C + Python to understand how attacker infrastructure actually works, because you can't defend against what you don't understand.

### [CSPLens](https://github.com/pratiksingh94/csplens) — CSP Analyzer & Builder
> Browser-based tool that grades Content Security Policies, flags dangerous patterns like `unsafe-inline`, and maps the XSS attack surface per source expression. Analysis engine is a clean Unix-style pipeline: `analyse(classify(parse(csp)))`.
